## ✨ Features
- 🔥 Real-time feed with post types
- 👥 Developer profiles & networking  
- 💬 Comments & likes system
- 📱 Mobile app waitlist
- 🏗️ Project showcases
- 🔍 Stack tag filtering

## 🛠️ Tech Stack
- **Frontend:** Next.js 14, TypeScript, Tailwind CSS
- **Backend:** Firebase Firestore, Firebase Auth
- **Deployment:** Vercel
- **Email:** EmailJS

## 🚀 Getting Started

```bash
git clone https://github.com/Keerthanreddy01/collabsphere-web
cd collabsphere-web/frontend
npm install
cp .env.example .env.local
# Fill in your Firebase + EmailJS keys
npm run dev
```
